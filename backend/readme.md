### Backend Setup
```
npm init -y
npm i express mongoose dotenv cloudinary cors @clerk/express socket.io
npm i -D nodemon
npm i express-fileupload
```
`"type": "module"` -> for import syntax instead of commonjs

Define api routes in "routes folder"; import in index.js
